CSS, XLink, XPointer и XInclude

Цели на упражнението:

  • Дефиниране на вграден (Inlining) CSS стил
  • Асоцииране на CSS дефиниция на стилове (вътрешна и външна) в XML и HTML документ
  • Дефиниране на CSS стилове върху различни медии (напр. принтер, екран, т.н.)
  • Използване на няколко CSS дефиниции за един XML/HTML документ
  • Вмъкване на CSS дефиниция в друга CSS дефиниция
  • Използване на CSS селектори - прости селектори, контекстови селектори, селектори и класове, различни типове селектори - пряк наследник, "брат/сестра", за атрибути и т.н.
  • Асоцииране на даден CSS стил с част/и от XML/HTML документ, чрез използване на ID, CLASS, STYLE
  • Групиране на CSS дефиниции
  • Каскаден ред
  • Кутиен модел - блокови (block) и поредови (inline) кутии
  • Позициониране
  • Използване на XLink, XPointer и XInclude
